
## Requirements
`pip install -r requirements.txt`

## Run
- Download pre-processed datasets from this [link](https://drive.google.com/drive/folders/18iqbsrh2f29kbRZ7SSfikjaLzTTDX0z3?usp=sharing) and put them in the data folder.
- ` python run_model.py`
